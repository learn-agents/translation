# AI Agents huggingface open [course](https://huggingface.co/learn/agents-course)

## Course

Вам нужно будет пройти Units 0, 1, 2, 2.1 [этого курса](https://huggingface.co/learn/agents-course)

**Units 2.2 и далее проходить не нужно!**

:::info инструкция

- первые модули, посвященные LLM, можно ЧАСТИЧНО пропускать, если вы знакомы с информацией из них
    - учтите, что в модулях со знакомым названием, где-то в середине/конце может быть полезная информация
- пользуйтесь `https://proxy.merkulov.ai` и вашим LLM API key [(что это?)](https://llm-keygen.merkulov.ai)
    - обращайтесь в прокси, инициализируя модели через OpenAI или LiteLLM классы в фреймворках(они есть в langchain, smolagents, llamaindex)

:::
