# Метаобучение

Метаобучение - это процесс обучения тому, как эффективно учиться. Это стратегия, которая позволяет:

1. Оптимизировать время обучения за счёт AI-инструментов
2. Выстраивать индивидуальную систему усвоения знаний
3. Автоматизировать рутинные задачи обучения
4. Поддерживать когнитивные ресурсы организма

### Чтение хэндбука

1. **Введение страницы и вопросы** нужно для того, чтобы сформировать "скелет", на который будет "навешиваться" дальнейшая теория/практика.
2. **Экста секции** не всегда обязательно читать.
3. **Заключение** нужно для того, чтобы еще раз быстро вспомнить только что изученный материал, таким образом совершив первое повторение.
4. **Упражнения** чаще всего представляют из себя список сложных вопросов. Во время трудных размышлений мозг активно генерирует нейронные связи, которые закрепляются в долгосрочной памяти.
5. **В комментариях** можно найти дополнительные ресурсы и советы от других студентов.

### Когнитивные техники

#### Используйте интерливинг (-15% от времени обучения)
- Чередуйте разные типы задач (чтение ↔ практика ↔ конспектирование)
- Сочетайте изучение теории с реальными кейсами из вашей работы
- Используйте технику "распределенного повторения" для закрепления материала

#### Практикуйте активное вспоминание
- После каждого раздела закрывайте материал и пытайтесь воспроизвести ключевые идеи
- Используйте флэш-карты с вопросами по пройденному материалу
- Объясняйте концепции вслух или письменно как будто преподаёте их новичку

#### Оптимизируйте контекст обучения
- Создавайте ментальные карты связей между концепциями с помощью AI-инструментов
- Группируйте информацию по смысловым блокам (chunking)
- Используйте технику Pomodoro (25/5 минут) для поддержания фокуса

#### Адаптируйте под свои биоритмы
- Определите пики продуктивности (утро/день/вечер)
- Сложные задачи планируйте на периоды максимальной концентрации
- Используйте "окна внимания" - 90-120 минутные блоки с перерывами

### ChatGPT Prompts

#### Шаблон сократического промпта
````
Ты — Сократ 2.0. Помоги мне разобраться с [ТЕМА/ПРОБЛЕМА] через диалог. Учти:
- Мой бэкграунд: [ОПИШИ СВОЙ ОПЫТ И УРОВЕНЬ]
- Предпочитаемый стиль: [АНАЛИТИЧЕСКИЙ/ПРАКТИЧЕСКИЙ/ВИЗУАЛЬНЫЙ]
- Область примеров: [ПРОГРАММИРОВАНИЕ/МАТЕМАТИКА/ДИЗАЙН и т.д.]

Правила:
1. Задавай не более 1 вопроса за раз
2. Чередуй типы вопросов (уточняющие, провокационные, альтернативные)
3. Предлагай "мини-эксперименты" для проверки гипотез
4. Используй аналогии из [УКАЖИ СФЕРУ]
5. При неправильном ответе — проси аргументировать позицию

Начни с: "Что для тебя самое непонятное в этой теме? С чего хочешь начать разбор?"
````

#### Помощь с решением leetcode задач
<details>
<summary>Помощь с leetcode задачами</summary>
```
Ты — наставник, который помогает ученику решать задачу с LeetCode. Используй сократический метод: не давай готовых ответов, а вместо этого веди ученика к решению через вопросы. Помогай понять условия задачи, навести на правильную стратегию, проанализировать крайние случаи, выбрать структуры данных и шаг за шагом подводи к решению. Приводи минимальные примеры, проси ученика объяснить свои рассуждения и проверяй их. Цель — развить мышление, а не просто решить задачу.

Начни с фразы: "Расскажи, как ты понял задачу, и какие у тебя есть первые идеи?"
```

#### Промпт для объяснения условия задачи или алгоритма:
```
Объясни ТЕМУ так, будто мне 5 лет. Используй простые слова, метафоры, аналогии из повседневной жизни (игрушки, конфеты, коробки, мультики). Избегай технических терминов, если только ты не объясняешь их очень просто. Представь, что ты рассказываешь сказку или играешь со мной в игру, и хочешь, чтобы я понял, что происходит.
```
