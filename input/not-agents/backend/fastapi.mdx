# FastAPI

Вам необходимо владеть минимум 1 бэкенд фреймворком: Django, Flask, FastAPI, NestJS, Spring Boot, Laravel, etc.

В этом разделе мы рассмотрим FastAPI из-за его популярности и простоты.
