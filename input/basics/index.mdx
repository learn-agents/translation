# Об этом блоке

## Что будет в блоке Basics

:::info
В блоке Basics мы сформируем фундамент, который будет необходим для изучения любых Applied AI  (GenAI, AI Agents, AgentOps) Engineering тем - модулей внутри данного handbook и информацией **за его пределами** - на других ресурсах.
:::

#### Основные темы:

- **Первый проект + бесплатные API ключи**: практическая реализация первого AI Агента с использованием бесплатных ресурсов
- **Что такое LLM**: знакомство с большими языковыми моделями, их принципами работы и возможностями
- **OpenAI API**: изучение интерфейса программирования приложений OpenAI, основные функции и методы взаимодействия с AI
- **Промпт-инжиниринг**: техники составления эффективных запросов к языковым моделям для получения нужных результатов

#### Дополнительные темы:
- **Метаобучение**: подходы к эффективному обучению и освоению новых технологий в сфере AI
- **Варианты использования LLM**: обзор практических сценариев применения языковых моделей в различных сферах
- **Как выбрать LLM**: критерии выбора подходящей языковой модели для конкретных задач
- **Vibe coding**: творческий подход к программированию с использованием AI

#### Легкая бэкенд-разработка:
- **Чанки**: работа с фрагментами данных и контекстом
- **Интернет**: подключение агентов к сети
- **Линукс**: базовые навыки работы в среде Linux
- **Базы данных**: хранение и обработка данных для AI Агентов
- **Микросервисы**: архитектура распределенных AI систем
- **Устойчивость**: создание стабильных и надежных AI решений

## Почему модули устроены так как устроены?

Каждый модуль - страница на данном сайте - состоит из следующих частей:

- **Введение и вопросы к модулю** - рекомендуется прочитать, чтобы создать в голове "скелет", на который в будущем будут навешиваться новые знания
- **Steps** - обязательный материал, база
- **Extra Steps** - дополнительные материалы, которые можно изучить, если интересно
- **Now we know** - подытог модуля - его полезно прочитать, чтобы мозг еще раз пробежался по материалу
- **Exercises** - сложные вопросы. Во время размышления над ними вам будет тяжело, но это нормально. Именно в этот момент мозг активно формирует новые нейронные связи.
