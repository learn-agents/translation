# Подготовка

## Минимальные требования

- Умение работать с VSCode или другой IDE
- Python на уровне синтаксиса и базовых библиотек.
    - [Инди курс по питону, 200к студентов](https://stepik.org/course/63085/promo#toc?search=4944556344)
    - [Добрый добрый питон, чилл курс (моя рекомендация)](https://stepik.org/course/100707/promo?search=6751006995)
    - [5 часовое видео by Александр Ильин](https://youtu.be/wDmPgXhlDIg?si=O5sfpcMhoKRglXZi)
    - а вообще рекомендую найти международный + бесплатный + интерактивный курс по питону, вернуться сюда и отредактировать этот текст
        - [Например этот питон курс, формирующий ЗАВИСИМОСТЬ](https://www.boot.dev/) (20$)

### Будет плюсом

- Базовые знания по программированию
- Базовые знания по машинному обучению и нейронным сетям
- Базовые знания по NLP и LLM

- Опыт легкой бэкенд разработки
